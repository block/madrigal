When starting a new project, always initialize the design system tokens before writing any component code. This ensures consistent spacing, typography, and color usage from the beginning.

1. Install the design token package
2. Import the base theme
3. Configure your build tool to resolve token aliases
4. Verify tokens load correctly in your dev environment
